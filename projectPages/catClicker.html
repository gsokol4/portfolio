This is a project that uses React, express.js and Node.js. This project was created as a way to combine these technologies. My favorite part of building this project was coming up with a method to make it rain cats on the main page. I created cat components that would generate a random cat image based on an array of images. I also used state to manage the position of the cat images and allow them to glide across the screen. The project also uses other react tools such as react router and useContext. The project is hosted on heroku and can be accessed there.
